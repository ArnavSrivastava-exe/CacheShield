# 🛡️ CacheShield — Smart API Cache Validator

CacheShield is a desktop application that monitors API response behavior in real-time and helps developers detect:

- ❌ Cache Misses
- ⚠️ Stale Data
- 🌀 Inefficient Caching
- 📉 Slow Response Times

Built with Python + Tkinter, and packaged as a Windows `.exe` app with PyInstaller.



## 🚀 Features

- ✅ Import and Export API list (JSON)
- 🟢 Start, Stop, and Monitor any API
- 📊 Real-time Graphs of Response Time
- 🔔 Alert System for Failing APIs
- 🧪 One-click API testing utility
- 🛠️ Built-in GUI (No terminal required)
